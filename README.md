# nocardreaderpay.com

SEO-first guide: how to take payments without a card reader. Funnels to [Valetfy](https://valetfy.com).

## Stack

- Astro (static, zero client JS)
- Tailwind CSS v4
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Domain

- Production: https://nocardreaderpay.com
- DNS via Porkbun → Vercel
